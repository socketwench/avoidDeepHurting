- name: Generate a timestamp
  set_fact:
    timestamp: "{{ lookup('pipe', 'date -I') }}"


- name: Config sync backup
  archive:
    src: "{{ sync_dir }}"
    dest: "{{ backup_dir }}/sync_{{ timestamp }}.tgz"
- name: backup database
  mysql_db:
    state: dump
    name: "{{ db_name }}"
    target: "{{ backup_dir }}/db_{{ timestamp }}.tgz"
    login_host: "{{ db_host }}"
    login_port: "{{ db_port | int }}"
    login_user: "{{ db_user }}"

  --executor ssh \

Takeaways

The Repo is the Server

Never SSH in unless something breaks

Everything is in the repo*

Use secure vars for logins and keys

Keep branches the same

Separate conf files per environment

Make CI do everything

Varnish ban, bounce Apache, anything needed

CI hurts now...

...so it doesn't later.
